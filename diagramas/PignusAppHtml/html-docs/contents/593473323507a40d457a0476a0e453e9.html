Registrar ingreso de persona

UMLUseCase
Pignus App :: Model :: Registrar ingreso de persona

Description

El Vigilante podrá escanear el código pdf417 de la cédula del visitante o bien registrarlo en un formulario. Y podrá tomar una foto del visitante.

NOTA: Al registrar un visitante se hará un chequeo automático de esta persona en la lista negra, y mostrará una alerta roja en caso de que esta persona esté en dicha lista
